51. Cara memperbaiki jika Winbox tidak mendeteksi MAC Address: 
1. Pastikan kabel LAN terhubung dengan benar.
2. Cek apakah interface Ethernet di komputer sudah aktif.
3. Pastikan firewall komputer tidak memblokir Winbox.
4. Coba restart Mikrotik dan cek MAC di "Neighbors" pada Winbox.
5. Jika masih tidak muncul, bisa gunakan IP statis untuk akses.
52. Pengertian VLAN dan jenis-jenisnya: 
VLAN (Virtual Local Area Network) adalah metode membuat beberapa jaringan logis dalam satu perangkat fisik.

Jenis-jenis VLAN:
- Default VLAN
- Data VLAN
- Voice VLAN
- Management VLAN
- Native VLAN
53. Masalah bandwidth Mikrotik tidak sesuai, solusinya: 
- Pastikan queue tree/queue simple sudah dikonfigurasi benar.
- Cek apakah ada traffic limiter di interface lain.
- Periksa apakah CPU Mikrotik overload.
- Gunakan "Torch" untuk memantau traffic.
- Update firmware Mikrotik agar performa optimal.
54. Langkah membuat DHCP Client-Server & Hotspot Mikrotik: 
1. DHCP Server:
   - IP → DHCP Server → Setup → Pilih interface → Atur IP Pool & Gateway → Apply.
2. DHCP Client:
   - IP → DHCP Client → Tambah → Pilih interface → Enable.
3. Hotspot:
   - IP → Hotspot → Hotspot Setup → Pilih interface → Masukkan IP → Atur IP Pool → Buat User Hotspot.
55. IP: 192.168.10.1/28 
a. Jumlah subnet: 16 subnet

c. Blok subnet: 192.168.10.0, 192.168.10.16, dst.
